PLACEHOLDER: Social Preview Image Required

This directory needs a homepage social preview image for LinkedIn, Twitter, and other social platforms.

Required file: llm-threatintel-home.png

Specifications:
- Size: 1200 x 630 pixels
- Format: PNG
- Background: dark / purple to match site theme
- Large readable text: "LLM ThreatIntel" and "GenAI Threat Intelligence"
- Optional subtitle: "Malicious LLM tools • AI malware • LLMjacking • Shadow AI"

This image will be used in Open Graph and Twitter Card metadata for social sharing previews.

Per-post images optional:
You can also add post-specific images as <post-slug>.png for better per-post sharing previews.
The build script will automatically use per-post images if present, falling back to homepage image.
